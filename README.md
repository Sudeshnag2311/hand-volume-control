# Hand Volume Control 🎚️

A hand gesture based volume control system using Python, OpenCV and MediaPipe.

---

## 🚀 Features
- Control system volume using hand gestures
- Real-time hand tracking
- Distance-based volume adjustment

---

## 🛠️ Technologies Used
- Python
- OpenCV
- MediaPipe
- PyAutoGUI
